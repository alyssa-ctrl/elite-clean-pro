03 / Honest disclosure

What we don't do.

Published, so there are no surprises. Some of these are available as add-ons; some are work we genuinely don't do. Either way, you know going in.

Move heavy furniture

We clean around. Lifting beds, sofas, or appliances is outside the standard scope.

Wash dishes

We clear dishes from the sink to enable counter access, then continue cleaning. We don't run loads. +$25 add-on

Laundry

Washing, drying, folding clothes or linens. Not part of standard residential.

Litter boxes

Pet waste handling. Available as a regular add-on if you'd prefer. +$15 add-on

Exterior windows

Interior only. Outside-the-glass cleaning needs a specialist crew, not ours.

Inside oven / fridge

Exteriors only on standard. Add deep clean for full interior treatment. +$80 deep add-on

Carpet shampoo / steam

We vacuum thoroughly. Wet extraction is a specialist service we refer to trusted partners.

Mold remediation

We clean visible surface mildew with BioSmart. Active mold remediation requires certification we don't carry.

Hoarding / junk removal

Cleaning happens after the space is accessible. We can refer specialists who handle this work.
